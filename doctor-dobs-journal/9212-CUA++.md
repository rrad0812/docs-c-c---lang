### D-Flat++ na radnoj površini

Prošlog meseca smo započeli projekat D-Flat++ prikazivanjem datoteka zaglavlja koje opisuju klase desktop i hardverskih uređaja. DF++ aplikacijski program počinje sa prozorom aplikacije na radnoj površini. Aplikacija instancira prozor aplikacije i povezuje funkcije člana komande menija i menija sa prozorom aplikacije. Prozor aplikacije se povezuje sa globalnom radnom površinom, koja sadrži objekte uređaja za korisnički interfejs. Ovog meseca ćemo pogledati .CPP izvorne datoteke koje sadrže funkcije članova za desktop i njegove uređaje.

[LISTING_ONE](#listing_one) je "desktop.cpp", koji sadrži kod za samu radnu površinu. Počinje uspostavljanjem logike rukovanja izuzetcima za uslove van memorije. Iako savremena C++ literatura definiše rukovanje izuzetcima, nijedan PC kompajler to još uvek ne implementira. Međutim, funkcija set_nev_handler je već neko vreme u C++ i obezbeđuje mesto za programe kada sistem ostane bez memorije. Borland C++ implementira set_nev_handler u skladu sa C++ tradicijom, ali Microsoft C++ koristi sopstveni nekonvencionalni format, tako da desktop.cpp počinje sa uslovom vremena kompajliranja da bi bio u skladu sa bilo kojim kompajlerom koji koristite. Od ovog pisanja, razvijam sa Borland kompajlerom, a zatim prenosim kod na Microsoft kompajler, koji je obično besprekorni port.

Svaka DF++ aplikacija počinje sa podrazumevanim objektom radne površine koji se na odgovarajući način zove „desktop“. Kada aplikacija proglasi prozor aplikacije, konstruktor prozora aplikacije se povezuje sa globalnim objektom radne površine. Definicija klase radne površine uključuje ugrađene objekte za uređaje na radnoj površini. Konstruktor klase radne površine postavlja podrazumevane pokazivače prozora na null i uspostavlja adresu obrađivača izuzetaka van memorije, koji izlazi iz programa. Zatim konstruktor kaže objektu kursora da se sakrije. Pošto je desktop objekat globalan, njegov konstruktor radi ispred glavne funkcije. Destruktor se pokreće kada se main vrati i kaže objektu kursora da se ponovo prikaže, pod pretpostavkom da se program pokreće iz komandne linije gde je kursor potreban i da će pojedinačni prozori dokumenta pokazati kursor ako im je potreban.

Metoda Desktop::DispatchEvents poziva metodu DispatchEvents za svaki objekat uređaja koji može da generiše poruke o događajima. Tastatura, miš i sat mogu da generišu takve događaje, tako da svaki ima funkciju da pošalje svoje događaje u bilo koji prozor koji ih primi.

Liste od dva do sedam su screen.cpp, mouse.cpp, cursor.cpp, keiboard.cpp, speaker.cpp i clock.cpp. Ovi moduli sadrže funkcije članova za klase uređaja definisane u datotekama zaglavlja od prošlog meseca.

Objekat ekrana. Datoteka screen.cpp (Dva lista, strana 154) upravlja izlazom na ekran kao i čitanjem i pisanjem video memorije. Konstruktor ekrana određuje tip prikaza i video režim, stranicu i memorijsku adresu. Doseže u BIOS RAM Data Area računara da bi dobio trenutnu visinu i širinu karaktera na ekranu. Ovaj kod, kao i kod ostalih modula ovog meseca, u velikoj meri zavisi od hardvera sistema. Ostatak DF++ je što je moguće nezavisniji od hardvera. U screen.cpp ima mnogo koda niskog nivoa. Modul direktno pristupa video memoriji i koristi funkciju INT86 za obavljanje video poziva BIOS-a.

Klasa uključuje funkcije članova za testiranje EGA ili VGA monitora, koje će odrediti formate ekrana koje možete izabrati. Postoji funkcija za pomeranje dela ekrana i funkcije za čitanje i pisanje pojedinačnih znakova video memorije na osnovu koordinata ekrana. Postoji i funkcija za pisanje stringa u video memoriju, koja koristi pomeranje bloka, a ne niz poteza karaktera. Ovo je strategija učinka. Kad god može, DF++ upisuje na ekran red po red.

Poslednje dve funkcije dobijaju i stavljaju video memoriju u velike blokove. Blokovi su opisani Rect objektima koji se prosleđuju kao parametri. Pretpostavlja se da su baferi držači za video memoriju koja može da stane u pravougaonike.

Objekat miša. Datoteka mouse.cpp [LISTING_THREE](#listing_three) sadrži kod za upravljanje objektom Mouse. Konstruktor utvrđuje da li je upravljački program miša instaliran gledajući vektor prekida miša. Ako je vektor 0 ili ako ukazuje na iret kod operacije na mašinskom jeziku, upravljački program miša nije pokrenut i program neće pokušati da koristi miš.

Konstruktor i destruktor miša čuvaju i vraćaju stanje miša. Nisam siguran koliko je to neophodno osim ako ne koristite DF++ za razvoj TSR-a. Zašto bi neko koristio C++ za pisanje TSR-ova, međutim, ne znam. To bi bilo kao da se voziš na posao u Konkordu. Ili Džordž Buš koji vodi kampanju sa zadnjeg dela voza. Ostale funkcije dobijaju i postavljaju poziciju kursora miša, prikazuju i sakrivaju kursor miša, određuju da li se miš pomerio od poslednjeg provera programa, vidi da li je dugme pritisnuto ili otpušteno i opisuju granice kretanja miša na ekranu. Funkcija DispatchEvent, koju poziva istoimena funkcija Desktop objekta, poziva funkcije za slanje događaja koji se odnose na pritiske i otpuštanje tastera i pokrete miša.

Funkcija MouseVindov određuje koji prozor treba da primi događaj miša. Ako je prozor uhvatio fokus, događaj ide u taj prozor bez obzira na poziciju kursora miša. U suprotnom, događaj ide u bilo koji prozor u kojem se nalazi kursor miša. Globalna funkcija u prozoru, u drugoj izvornoj datoteci, vraća pokazivač prozora dat skupu koordinata. On određuje prozor posmatrajući njegovu poziciju u odnosu na druge prozore koji dele iste koordinate, ali koji su prekriveni. Prvi prozor je onaj vraćen.

Funkcija DispatchLeftButton kontroliše ponašanje miša slično tipu. Ako korisnik neko vreme drži dugme miša pritisnuto, program šalje ponovljene događaje levog dugmeta u prozor. Funkcija DispatchRelease ne samo da šalje događaj oslobađanja, već i postavlja tajmer. Ako se drugi događaj oslobađanja dogodi pre nego što tajmer istekne, funkcija šalje događaj dvostrukog klika. Funkcija DispatchMove šalje događaj pomeranja ako je kursor miša u poziciji koja nije ona gde ga je program poslednji put pogledao.

Kursorski objekat. Cursor.cpp [LISTING_FOUR](#listing_four) sadrži kod za prikaz, sakrivanje, pomeranje, čitanje i veličinu kursora tastature na ekranu. Ovo su sve standardne operacije BIOS-a osim čuvanja i vraćanja kursora. Njegov oblik i stanje vidljivosti su na gomili. Prozori koji koriste kursor čuvaju kursor pre nego što bilo šta urade sa njim i vraćaju ga kada završe. Pošto se ovaj proces može desiti u ugnežđenim prozorima, stek obezbeđuje da se kursor vrati u svoju ispravnu konfiguraciju kako se različiti prozori otvaraju i zatvaraju, preuzimajući i oslobađajući fokus.

Objekat tastature. Keiboard.cpp [LISTING_FIVE](#listing_five) sadrži funkcije člana za objekat Keiboard. Njegova funkcija DispatchEvents određuje koji prozor treba da dobije događaj sa tastature koristeći sledeća pravila prioriteta. Ako je prozor uhvatio fokus, taj prozor dobija događaj. U suprotnom, događaj ide u prozor koji ima fokus, ako postoji. Ako nijedan prozor nema fokus, događaj ide u prozor aplikacije.

Objekat tastature šalje dva događaja, jedan kada korisnik pritisne taster i jedan kada se promeni status promene. Klasa takođe obezbeđuje funkciju AltConvert za pretvaranje tastera Alt+ u njihov ekvivalent ASCII slova ili brojeva.

Govornik objekat. Speaker.cpp [LISTING_SIX](#listing_six) sadrži funkciju Beep za stvaranje zvuka kroz audio sistem računara. Funkcija pokreće sistemski zvučnik tako što programira frekvenciju, pokreće zvuk i ostavlja ga uključenim tokom dva otkucaja sistemskog sata. Rezultirajuće zujanje koristi DF++ da kaže korisniku da izbor nije važeći.

Objekat sata. Clock.cpp [LISTING_SEVEN](#listing_seven) šalje događaj clock-tick u prozor aplikacije svakih 19 otkucaja sistemskog intervalnog tajmera. Ovo je približno jedan događaj u sekundi, ali nije precizno jer se otkucaji tajmera javljaju 18,2 puta u sekundi. Svrha događaja je da omogući statusnu liniju prozora aplikacije da prikaže doba dana bez potrebe da se nepotrebno ispituje sistemski sat vremena u danu.

Screen::Screen()
{
    if (isEGA() || isVGA())    {
        // --- turn blinking off
        regs.x.ax = 0x1003;
        regs.h.bl = 0;
        int86(VIDEO, &regs, &regs);
    }
    // ---- get the video mode and page
    regs.h.ah = 15;
    int86(VIDEO, &regs, &regs);
    mode = regs.h.al;
    page = regs.x.bx;
    page &= 0xff00;
    mode &= 0x7f;
    // ---- Monochrome Display Adaptor or text mode
    if (isMono())
        address = 0xb000;
    else
        // ------ Text mode
        address = 0xb800 + page;
    width = *(unsigned char far *)( MK_FP(0x40,0x4a) );
    if (isVGA() || isEGA())
        height = *(unsigned char far *)( MK_FP(0x40,0x84) )+1;
    else
        height = 25;
}
// ---- test for EGA
Bool Screen::isEGA(void)
{
    if (isVGA())
        return False;
    regs.h.ah = 0x12;
    regs.h.bl = 0x10;
    int86(VIDEO, &regs, &regs);
    return (Bool) (regs.h.bl != 0x10);
}

Bool Screen::isVGA(void)
{
    regs.x.ax = 0x1a00;
    int86(VIDEO, &regs, &regs);
    return (Bool) (regs.h.al == 0x1a && regs.h.bl > 6);
}

void Screen::Scroll(Rect &rc, int d, int fg, int bg)
{
    desktop.mouse().Hide();
    regs.h.cl = rc.Left();
    regs.h.ch = rc.Top();
    regs.h.dl = rc.Right();
    regs.h.dh = rc.Bottom();
    regs.h.bh = clr(fg,bg);
    regs.h.ah = 7 - d;
    regs.h.al = 1;
    int86(VIDEO, &regs, &regs);
    desktop.mouse().Show();
}

void Screen::WriteVideoString(char *s,int x,int y,int fg,int bg)
{
    if (x < width && y < height)    {
        int len = strlen(s);
        int *ln = new int[len];
        int *cp1 = ln;
        int col = clr(fg,bg) << 8;
        while (*s)    {
            *cp1++ = (*s & 255) | col;
            s++;
        }
        if (x + len >= width)
            len = width - x;
        desktop.mouse().Hide();
        movedata(FP_SEG(ln),FP_OFF(ln),address,vad(x,y),len*2);
        desktop.mouse().Show();
        delete [] ln;
    }
}
// -- read a rectangle of video memory into a save buffer
void Screen::GetBuffer(Rect &rc, char *bf)
{
    if (rc.Left() >= width || rc.Top() >= height)
        return;
    int ht = rc.Bottom()-rc.Top()+1;
    int bytes_row = (rc.Right()-rc.Left()+1) * 2;
    unsigned vadr = vad(rc.Left(), rc.Top());
    desktop.mouse().Hide();
    while (ht--)    {
        movedata(address, vadr, FP_SEG(bf),
                FP_OFF(bf), bytes_row);
        vadr += width*2;
        bf = (char far *)bf + bytes_row;
    }
    desktop.mouse().Show();
}

#include <stdio.h>
#include "desktop.h"

// -------- mouse constructor
Mouse::Mouse()
{
    // ------- see if mouse driver is installed
    unsigned char far *ms;
    ms = (unsigned char far *)
        MK_FP(peek(0, MOUSE*4+2), peek(0, MOUSE*4));
    // --- if the interrupt vector is null or points to a retf,
    //     the mouse driver is not installed
    installed = (Bool) (ms != NULL && *ms != 0xcf);

    if (installed)    {
        // --- get the mouse state buffer size
        CallMouse(BUFFSIZE);
        statebuffer = new char[regs.x.bx];
        // --- save the mouse state
        CallMouse(SAVESTATE, 0, 0,
            FP_OFF(statebuffer), FP_SEG(statebuffer));
        // --- reset the mouse
        CallMouse(RESETMOUSE);
        prevx = prevy =
        clickx = clicky =
        releasex = releasey -1;
        SetTravel(0, desktop.screen().Width()-1, 0,
                                                  desktop.screen().Height()-1);
    }
}
Mouse::~Mouse()
{
    if (installed)    {
        Hide();
        // --- restore the mouse state
        CallMouse(RESTORESTATE, 0, 0,
            FP_OFF(statebuffer), FP_SEG(statebuffer));
        delete [] statebuffer;
    }
}
void Mouse::GetPosition(int &mx, int &my)
{
    mx = my = 0;
    if (installed)    {
        CallMouse(READMOUSE);
        mx = regs.x.cx/8;
        my = regs.x.dx/8;
        if (desktop.screen().Width() == 40)
            mx /= 2;
    }
}
void Mouse::SetPosition(int x, int y)
{
    if (installed)    {
        if (desktop.screen().Width() == 40)
            x *= 2;
        CallMouse(SETPOSITION,0,x*8,y*8);
    }
}
Bool Mouse::Moved()
{
    int x, y;
    Bool rtn = False;
    if (installed)    {
        GetPosition(x, y);
        rtn = (Bool) (x != prevx || y != prevy);
        prevx = x;
        prevy = y;
    }
    return rtn;
}
void Mouse::Show()
{
    if (installed)
        CallMouse(SHOWMOUSE);
}
void Mouse::Hide()
{
    if (installed)
        CallMouse(HIDEMOUSE);
}
Bool Mouse::LeftButton()
{
    Bool rtn = False;
    if (installed)    {
        CallMouse(READMOUSE);
        rtn = (Bool) ((regs.x.bx & 1) == 1);
    }
    return rtn;
}
Bool Mouse::ButtonReleased()
{
    Bool rtn = False;
    if (installed)    {
        CallMouse(BUTTONRELEASED);
        rtn = (Bool) (regs.x.bx != 0);
    }
    return rtn;
}
void Mouse::SetTravel(int minx, int maxx, int miny, int maxy)
{
    if (installed)    {
        if (desktop.screen().Width() == 40)    {
            minx *= 2;
            maxx *= 2;
        }
        CallMouse(XLIMIT, 0, minx*8, maxx*8);
        CallMouse(YLIMIT, 0, miny*8, maxy*8);
    }
}
void Mouse::CallMouse(int m1,int m2,int m3,int m4, unsigned es)
{
    struct SREGS sregs;
    segread(&sregs);
    if (es != 0)
        sregs.es = es;
    regs.x.dx = m4;
    regs.x.cx = m3;
    regs.x.bx = m2;
    regs.x.ax = m1;
    int86x(MOUSE, &regs, &regs, &sregs);
}
// ------ get the window to send mouse events
DFWindow *Mouse::MouseWindow(int mx, int my)
{
    DFWindow *Mwnd;
    if (desktop.FocusCapture() != NULL)
        Mwnd = desktop.FocusCapture();
    else
        Mwnd = inWindow(mx, my);
    return Mwnd;
}
void Mouse::DispatchRelease()
{
    if (ButtonReleased())    {
        int mx, my;
        GetPosition(mx, my);
        DFWindow *Mwnd;
        if ((Mwnd = MouseWindow(mx, my)) == NULL)
            return;
        // ------- disable typematic check
        clickx = clicky = -1;
           delaytimer.DisableTimer();
        // ------- the button was released
        if (mx == releasex && my == releasey)    {
            // ---- same position as last left button release
            if (doubletimer.TimerRunning())    {
                // -- second click before double timeout
                doubletimer.DisableTimer();
                Mwnd->DoubleClick(mx, my);
                releasex = releasey = -1;
            }
        }
        else    {
            doubletimer.SetTimer(DOUBLETICKS);
            Mwnd->ButtonReleased(mx, my);
               releasex = mx;
               releasey = my;
        }
    }
}
void Mouse::DispatchMove()
{
    if (Moved())    {
        int mx, my;
        GetPosition(mx, my);
        DFWindow *Mwnd;
        if ((Mwnd = MouseWindow(mx, my)) != NULL)    {
            Mwnd->MouseMoved(mx, my);
            clickx = clicky = -1;
        }
    }
}
void Mouse::DispatchLeftButton()
{
    if (LeftButton())    {
        int mx, my;
        GetPosition(mx, my);
        DFWindow *Mwnd;
        if ((Mwnd = MouseWindow(mx, my)) == NULL)
            return;
        if (mx == clickx && my == clicky)    {
            if (delaytimer.TimedOut())    {
                // ---- button held down a while
                delaytimer.SetTimer(DELAYTICKS);
                // ---- post a typematic-like button
                Mwnd->LeftButton(mx, my);
            }
        }
        else    {
            // --------- new button press
            delaytimer.SetTimer(FIRSTDELAY);
            if (Mwnd->SetFocus())
                Mwnd->LeftButton(mx, my);
            clickx = mx;
               clicky = my;
        }
    }
}
// -------- dispatch mouse events
void Mouse::DispatchEvent()
{
    DispatchRelease();
    DispatchMove();
    DispatchLeftButton();
}

#include "cursor.h"
#include "desktop.h"

Cursor::Cursor()
{
    cs = 0;
    Save();

Cursor::~Cursor()
{
    Restore();

// ------ get cursor shape and position
void Cursor::GetCursor()
{
    regs.h.ah = READCURSOR;
    regs.x.bx = desktop.screen().Page();
    int86(VIDEO, &regs, &regs);
}
// -------- get the current cursor position
void Cursor::GetPosition(int &x, int &y)
{
    GetCursor();
    x = regs.h.dl;
    y = regs.h.dh;
}
// ------ position the cursor
void Cursor::SetPosition(int x, int y)
{
    regs.x.dx = ((y << 8) & 0xff00) + x;
    regs.h.ah = SETCURSOR;
    regs.x.bx = desktop.screen().Page();
    int86(VIDEO, &regs, &regs);
}
// ------ save the current cursor configuration
void Cursor::Save()
{
    if (cs < MAXSAVES)    {
        GetCursor();
        cursorshape[cs] = regs.x.cx;
        cursorpos[cs] = regs.x.dx;
        cs++;
    }
}
// ---- restore the saved cursor configuration
void Cursor::Restore()
{
    if (cs)    {
        --cs;
        regs.x.dx = cursorpos[cs];
        regs.h.ah = SETCURSOR;
        regs.x.bx = desktop.screen().Page();
        int86(VIDEO, &regs, &regs);
        SetType(cursorshape[cs]);
    }
}
/* ---- set the cursor type ---- */
void Cursor::SetType(unsigned t)
{
    regs.h.ah = SETCURSORTYPE;
    regs.x.bx = desktop.screen().Page();
    regs.x.cx = t;
    int86(VIDEO, &regs, &regs);
}
/* ----- swap the cursor stack ------- */
void Cursor::SwapStack()
{
    if (cs > 1)    {
        swap(cursorpos[cs-2], cursorpos[cs-1]);
        swap(cursorshape[cs-2], cursorshape[cs-1]);
    }
}
/* ------ hide the cursor ------ */
void Cursor::Hide()
{
    GetCursor();
    regs.h.ch |= HIDECURSOR;
    regs.h.ah = SETCURSORTYPE;
    int86(VIDEO, &regs, &regs);
}
/* ------ show the cursor ------ */
void Cursor::Show()
{
    GetCursor();
    regs.h.ch &= ~HIDECURSOR;
    regs.h.ah = SETCURSORTYPE;
    int86(VIDEO, &regs, &regs);
}

int Keyboard::GetKey()
{
    int c;
    while (KeyHit() == False)
        ;
    if (((c = bioskey(0)) & 0xff) == 0)
        c = (c >> 8) | 0x1080;
    else
        c &= 0xff;
    return c & 0x10ff;
}
/* ---------- read the keyboard shift status --------- */
int Keyboard::GetShift()
{
    regs.h.ah = 2;
    int86(KEYBRD, &regs, &regs);
    return regs.h.al;
}
/* ------ convert an Alt+ key to its letter equivalent ----- */
int Keyboard::AltConvert(int c)
{
    int i, a = 0;
    for (i = 0; i < 36; i++)
        if (c == altconvert[i])
            break;
    if (i < 26)
        a = 'a' + i;
    else if (i < 36)
        a = '0' + i - 26;
    return a;
}
Bool Keyboard::ShiftChanged()
{
    int sk = GetShift();
    Bool rtn = (Bool) (sk != shift);
    shift = sk;
    return rtn;
}

{
    // ---- find window for keyboard events
    DFWindow *Kwnd = desktop.FocusCapture() ?
                     desktop.FocusCapture() :
                     desktop.InFocus() ?
                     desktop.InFocus() : desktop.ApplWnd();
    if (ShiftChanged())
           // ---- the shift status changed
        Kwnd->ShiftChanged(GetShift());
    if (KeyHit())
        // --- a key was pressed
        Kwnd->Keyboard(GetKey());
}

void Speaker::Beep()
{
    outp(0x43, 0xb6);               // program the frequency
    outp(0x42, (int) (COUNT % 256));
    outp(0x42, (int) (COUNT / 256));
    outp(0x61, inp(0x61) | 3);      // start the sound
    // -------- wait two clock ticks
    const int far *clk = (int far *) MK_FP(0x40,0x6c);
    int then = *clk+2;
    while (*clk < then)
        ;
    outp(0x61, inp(0x61) & ~3);     // stop the sound
}
